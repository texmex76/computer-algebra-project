\begin{abstract}\noindent We investigate the basic theory of the modular method for GCD computations. Starting with a basic algorithm for integers, we transform it into a modular one. Furthermore, we dive into a modular algorithm to compute the GCD of univariate and multivariate polynomials. We implemented all algorithms and tested them. We carried out performance measurements on the modular integer GCD algorithm and compared it with the basic one.
\end{abstract}
